# 🖤 Vibeke Stene Tribute Landing Page

Uma landing page criada em homenagem à cantora norueguesa Vibeke Stene, conhecida mundialmente por sua trajetória no metal sinfônico e gothic metal.

![alt text](<FireShot Capture 033 - Vibeke Stene Fan - [127.0.0.1].png>)

## 🌑 Sobre o Projeto

Este projeto foi desenvolvido com foco em experiência visual imersiva, inspirado em websites modernos no estilo Awwwards, utilizando efeitos de scroll, parallax, atmosfera gótica e animações suaves.

A proposta da landing page é transmitir elegância, melancolia e profundidade artística através de uma interface dark e cinematográfica.

## ⚡ Funcionalidades

### 🎬 Hero Section Cinemática

Banner fullscreen com composição visual atmosférica, overlay escuro e efeito de profundidade inspirado em páginas modernas de portfólio.

### 🌫️ Efeito Parallax

Seções com imagem fixa ao fundo que acompanham o scroll da página, criando sensação de imersão e movimento.

### ✨ Reveal Animations

Elementos aparecem suavemente conforme o usuário navega pela landing page.

### 🖱️ Cursor Customizado

Cursor glow estilizado em tons de roxo e branco para reforçar a identidade visual dark.

### 🕯️ Atmosfera Gótica

Uso predominante de:
- Preto profundo
- Tons de cinza escuro
- Iluminação cinematográfica
- Contrastes suaves
- Sombras dramáticas

### 🖼️ Galeria Interativa

Imagens com:
- Hover 3D
- Zoom suave
- Efeito de profundidade
- Transições cinematográficas

### 🌌 Partículas Flutuantes

Sistema de partículas animadas em background para criar ambientação etérea.
